# 中证1000技术面量化选股系统

## 项目概述
本项目旨在构建一个基于技术面指标的量化选股系统，针对中证1000指数成分股进行分析。

## 系统设计

### 1. 数据获取模块
- 使用akshare获取中证1000最新成分股
- 获取股票历史行情数据（日线）
- 数据清洗和预处理

### 2. 技术指标计算模块
- MA（移动平均线）：5日、10日、20日、60日
- RSI（相对强弱指数）：14日周期
- MACD（指数平滑异同移动平均线）
- 布林带（Bollinger Bands）：20日周期，2倍标准差

### 3. 策略信号生成模块
- MA金叉策略：短期MA上穿长期MA
- RSI超卖策略：RSI < 30
- MACD金叉策略：DIF上穿DEA
- 布林带策略：价格触及下轨
- 综合评分系统

### 4. 回测分析模块
- 多周期历史回溯
- 绩效指标计算
- 结果可视化

### 5. 输出模块
- Markdown报告
- CSV结果文件
- 可视化图表

## 参数设置
```python
# 技术指标参数
MA_PERIODS = [5, 10, 20, 60]
RSI_PERIOD = 14
BB_PERIOD = 20
BB_STD = 2
MACD_FAST = 12
MACD_SLOW = 26
MACD_SIGNAL = 9

# 策略权重
STRATEGY_WEIGHTS = {
    'ma_cross': 0.3,
    'rsi_oversold': 0.25,
    'macd_cross': 0.25,
    'bb_touch': 0.2
}
```

## 选股逻辑
1. 计算各项技术指标
2. 生成买卖信号
3. 综合评分排序
4. 选择评分最高的前20只股票
5. 风险分散：行业分布均衡
